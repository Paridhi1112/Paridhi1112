🔗 [GitHub Repo](https://github.com/Paridhi1112/claimarmor-ai)

**Key Features:**
- 🧩 Multi-agent orchestration via ADK's native `sub_agents` routing to establish a strict execution graph
- 🔒 Secure MCP server integrating `@modelcontextprotocol/server-filesystem` for sandboxed, read-only file access
